Purpose: This folder is for URM batch jobs ready to be tested on the server. 

For each job, create a folder that includes:
1. A README.txt which speficying:
	- Class.Method(param): the class, method, and any paramaters that is needed to run the job 
	- Run: how frequently the job needs to be ran.
	- Description: what data the job uses, what is meant to be accomplished, and any extra clarification of paramaters if necessary.
	- Java Version: what version of java was used to create/run the job 
2. The exported jar file.
3.Any referenced libraries needed to compile/run the job
